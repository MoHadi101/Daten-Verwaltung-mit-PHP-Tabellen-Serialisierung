# PHP Datenverwaltung mit Serialisierung

Eine einfache CRUD-Anwendung (Create, Read, Update, Delete) in PHP, die Daten in einer Textdatei mittels Serialisierung speichert. Ideal für Lernzwecke und kleine Projekte ohne Datenbank.

## 📋 Features

- ✅ **Create** - Neue Datensätze hinzufügen
- 📖 **Read** - Alle Datensätze in einer Tabelle anzeigen
- ✏️ **Update** - Vorhandene Datensätze bearbeiten
- 🗑️ **Delete** - Datensätze löschen
- 💾 **Persistente Speicherung** in einer Textdatei (`dummy.txt`)
- 🔄 **Auto-Increment IDs** - Automatische ID-Vergabe
- 🛡️ **XSS-Schutz** durch `htmlspecialchars()`
